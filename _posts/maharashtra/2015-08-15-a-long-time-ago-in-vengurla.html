Five years ago, a few of us drove from Goa to Vengurla, a small beachside town in Sindhudurg, Maharashtra.

{% include image.html loc = "https://farm6.staticflickr.com/5763/20373726310_f1f11f017b_h.jpg" alt = "Approaching the Vengurla beach" %}

It would take you a couple of hours to reach the Vengurla beach, depending on where you depart from, and how fast you intend to drive. It is around an hour's drive from Mapusa.

{% include image.html loc = "https://farm6.staticflickr.com/5727/20568220461_6f399727ce_h.jpg" alt = "Happy Dog" %}

Borders crossed, to the beach it was! We were greeted by some of the local strays, who were all too happy to see us. A bow wow wow here, a bow wow wow there.

{% include image.html loc = "https://farm6.staticflickr.com/5750/19940844033_4ad229e8d9_h.jpg" alt = "Vengurla Beach" %} {% include image.html loc = "https://farm6.staticflickr.com/5671/20561722295_f23d297823_h.jpg" alt = "Vengurla Beach" %}

We were caught off guard, by a sudden downpour.

{% include image.html loc = "https://farm6.staticflickr.com/5633/20552779552_d1d49d29e4_h.jpg" alt = "Vengurla Beach" %}

But then there was the smell of wet mud. Wet mud.

{% include image.html loc = "https://farm6.staticflickr.com/5679/20552774702_e00d0df347_h.jpg" alt = "A dog at the Vengurla Beach" %}

There were more strays. Some came close.

{% include image.html loc = "https://farm6.staticflickr.com/5770/19939092224_6721069a2b_h.jpg" alt = "A dog at the Vengurla Beach" %}

While others stayed away, giving us their looks of disapproval.

{% include image.html loc = "https://farm6.staticflickr.com/5639/19939086894_d8d7f29a64_h.jpg" alt = "Poker at Vengurla" %}

The afternoon was spent playing poker.

{% include image.html loc = "https://farm6.staticflickr.com/5668/20568191151_d4678d0988_h.jpg" alt = "Vengurla Beach" %}

The air smelt fresh.

{% include image.html loc = "https://farm6.staticflickr.com/5812/20374982969_09bcc562e1_h.jpg" alt = "Vengurla Beach" %}

The waves crashed in the distance.

{% include image.html loc = "https://farm6.staticflickr.com/5812/20374982969_09bcc562e1_h.jpg" alt = "Vengurla Beach" %}

Footprints were left.

{% include image.html loc = "https://farm6.staticflickr.com/5793/20373665710_6f7dad5d6f_h.jpg" alt = "Vengurla Beach" %}

The tides moved back.

{% include image.html loc = "https://farm6.staticflickr.com/5707/20568156291_a5bb9849be_h.jpg" alt = "Vengurla Beach" %} {% include image.html loc = "https://farm6.staticflickr.com/5622/19940773593_d10848458f_h.jpg" alt = "Vengurla Beach" %} {% include image.html loc = "https://farm6.staticflickr.com/5760/20561646475_f7c6b2f934_h.jpg" alt = "Vengurla Beach" %}

Just another evening at the Vengurla beach.
